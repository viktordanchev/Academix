# Academix
Academix is ​​an electronic diary that completely replaces the paper version.
The main goal is to simplify and automate a large part of teachers duties.

# Tech Stack
- .NET Core 6
- ASP.NET Core MVC
- Entity Framework Core
- HTML
- CSS
- JavaScript
- MS SQL
